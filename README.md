# 图书管理系统 java860

## 项目概述

本系统是基于JSP+Servlet技术开发的图书管理系统，旨在为学校师生提供便捷的图书查询与借阅服务，同时为图书管理人员提供全面的管理功能。

## 技术栈

- JSP（JavaServer Pages）
- Servlet
- Java

## 功能模块

### 前台页面

- 公告显示：展示图书管理人员发布的最新公告信息。
- 图书查询：供师生查询感兴趣的图书。
- 留言建议：收集用户对图书管理提出的意见和建议。
- 个人功能（登录后）：
  - 个人资料修改
  - 个人借阅信息查询
  - 个人违章信息查询

### 后台管理

#### 图书管理人员功能

- 借阅图书
- 归还图书
- 借书记录显示与查找
- 还书记录显示与查找
- 公告增删改查

#### 系统管理员功能

- 对书籍的增删改查
- 对书籍分类的增删改
- 对借阅证的增删改查
- 对借阅信息的显示与查找
- 对借阅规则的增删改
- 对图书管理员的增删改
- 近期借阅书籍数量的折线图显示
- 表格数据Excel输出

## 系统角色

- 用户（普通师生）
- 图书管理员
- 系统管理员

## 运行环境

- Java运行环境
- Tomcat服务器
